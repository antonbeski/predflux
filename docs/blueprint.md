# **App Name**: PREDFLUX

## Core Features:

- Data Aggregation: Scrape news and financial data from various online sources (yfinance, etc.) for NSE and BSE stocks only.
- AI-Powered Stock Analysis: Analyze scraped data using Gemini 2.0 Flash only to determine buy, sell, and hold recommendations for each stock.
- Daily Recommendation Output: Generate a daily report containing stock recommendations (buy/sell/hold) based on AI analysis, designed to act as a trading tool.
- Secure API Key Management: Implement secure storage of the Gemini 2.0 Flash API key to prevent exposure, especially during Vercel deployment.
- Stock Details Display: Present comprehensive stock data, including price, news sentiment, and AI recommendations in a well-structured format.
- Recommendation History: Store the history of each recommendation for future reference.
- User Interface: Glassmorphic design, day/night toggle switch, data presented in tables, font adjusted to readability, interactive charts and make it mobile responsive completely

## Style Guidelines:

- Primary color: Light gray (#D3D3D3) to give a glassmorphic feel with good visibility.
- Background color: Almost white (#FAFAFA) with a low opacity.
- Accent color: Dark gray (#4A4A4A) to offer high contrast and clear reading in low opacity layers of glassmorphism.
- Font: 'Inter' (sans-serif) for a modern, clean and readable style for both titles and body text.
- Simple, outlined icons for stock actions (buy, sell, hold) to align with the minimal black and white aesthetic.
- Use a grid-based layout for responsive design, focusing on readability and information hierarchy with frosted glass panels.
- Subtle transition animations for stock data updates and AI analysis results.
